Place the ole folder from this php scripts folder into:
.../wamp/www/


You may use the .bat to run a script that will copy the 
www folder content into the this folder.


batch file contains this:

xcopy "C:\wamp64\www\ole" "D:\Ole2018\app\PhpScripts\ole" /Y /H /E /F /I
exit

xcopy "from folder to copy" "to folder to copy - Ole2018\app\PhpScripts\ole"
